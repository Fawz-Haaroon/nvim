CORE PRIMITIVES (editing, movement, modes)                            *core*

Modes
- Normal ........ Escape, jj, jk
- Insert ........ i, a, I, A, o, O
- Visual ........ v (char), V (line), Ctrl-v (block), gv reselect
- Command ....... : (colon)
- Terminal ...... in terminal buffers; exit to Normal: Ctrl-\ Ctrl-n or <Esc><Esc> or custom Ctrl-e

Undo/Redo and time travel
- Undo change ..................... u
- Redo change ..................... Ctrl-r
- Older text state ................ g-
- Newer text state ................ g+
- Jump back in time ............... :earlier 10s (supports s/m/h/f)
- Jump forward in time ............ :later 10s
- Repeat last command ............. .

Movement and motions
- Basic ........................... h j k l
- Words ........................... w b e ge
- Lines ........................... 0 ^ $
- Paragraphs ...................... { }
- Sentences ....................... ( )
- Find char forward/back .......... f{char} / F{char}; till t/T
- Search forward/back ............. /pattern, ?pattern; repeat n / N (centered)
- Start/end of line (custom) ...... H → ^, L → $
- Soft-wrapped lines (custom) ..... j → gj, k → gk when no count

Text objects (usable with y/d/c/v etc.)
- Words ........................... aw / iw
- Sentences ....................... as / is
- Paragraphs ...................... ap / ip
- Quotes/Brackets/Braces .......... a" i" a' i' a( i( a[ i[ a{ i{ ...

Operators
- Yank (copy) ..................... y, yy, Y
- Delete (cut) .................... d, dd, D
- Change (cut+insert) ............. c, cc, C
- Indent left/right (visual) ...... < / > (keeps selection)
- Join lines ...................... J (adds space), gJ (no space)
- Replace char / Replace mode ..... r / R
- Substitute char/line ............ s / S
- Case ............................ ~ toggle; gu/gU motions; guu/gUU line

Registers and clipboard
- System clipboard (custom) ....... <leader>y (motions/visual)
- System clipboard (line) ......... <leader>Y
- Paste from system clipboard ..... <leader>p
- Delete without yanking (bhole) .. x (char), <leader>d (motions/visual)
- Paste over selection (no pollute) p in Visual does "_dP
- Yank history (Yanky) ............ <leader>cy (Telescope yank_history)

Numbers and increment/decrement
- Increment / Decrement (custom) .. = / - mapped to Ctrl-a / Ctrl-x (also native Ctrl-a / Ctrl-x)
- Dial.nvim (extra inc/dec) ....... Normal: Ctrl-a / Ctrl-x; Visual: Ctrl-a / Ctrl-x; gCtrl-a / gCtrl-x

Search highlight
- Clear search highlight (custom) . <leader>cs or <Esc>

KEYMAP PRECEDENCE AND CONFLICTS                                    *precedence*

- <leader>1..4 Harpoon vs Buffer vs Tabs
  Your configuration defines overlapping uses for <leader>1..N across components.
  Final, effective mapping order at startup:
  1) Comfy Tabline maps <leader>1..9 to buffer selection.
  2) Core maps <leader>1..9 to tab selection (i gt), but these are older and later overridden.
  3) Functional keymaps map <leader>1..4 to Harpoon list select(1..4) after plugins load.
  Effective outcome:
  - <leader>1..4 → Harpoon select 1..4 (overrides buffer/tab mappings for 1..4)
  - <leader>5..9 → Comfy Tabline buffer 5..9 (remain active)

- Ctrl-i jumplist vs buffer tabline
  Core repurposed Ctrl-i for buffer next in tabline (custom), and provided <C-\> as a passthrough for jumplist forward.
  - Jump forward in jumplist ...... <C-\> or <leader>ji
  - Jump backward in jumplist ..... <leader>jo (Ctrl-o)

- Terminal exits and Ctrl-e
  Multiple exits exist: Ctrl-\ Ctrl-n (native), <Esc><Esc> (custom), and Ctrl-e (custom). All are safe; use any.

- Telescope preview toggles
  Telescope includes multiple preview toggles; current mappings rely on built-in defaults and some custom ones (see telescope-fixed.lua). Use ? inside Telescope for live help.
